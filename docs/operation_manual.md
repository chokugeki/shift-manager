# 運用マニュアル

## 1. はじめに
本マニュアルは「チャート式シフト管理システム」の操作方法を説明します。
本システムを使用することで、月間シフトの作成や日々の業務割り当てを効率的に行うことができます。

## 2. 初期設定
システムを使用する前に、必要なマスタ情報を登録します。

### 2.1 職員の登録
1. トップページから「職員登録」をクリックします。
2. 「新規追加」ボタンを押し、職員名と職種を入力して「保存」します。
3. 必要な人数分繰り返します。

### 2.2 業務内容の登録
1. トップページから「業務内容登録」をクリックします。
2. 「新規追加」ボタンを押し、業務名（例：入浴介助）、表示色、所要時間（分）を設定して「保存」します。
3. よく使う業務を一通り登録します。

## 3. 月間シフトの作成
毎月のシフトを作成する手順です。

1. トップページから「シフト作成・管理」をクリックします。
2. 画面上部の「前月」「次月」ボタンで対象月を表示します。
3. 表の中のプルダウンメニューから、各職員・各日のシフト（日勤、早番、公休など）を選択します。
   - **ヒント**: 休日希望が入力されている箇所にはマークが表示されます。
4. 画面下部の集計行で、日ごとの人員配置数が十分か確認します（不足時は赤字等で警告される場合があります）。

## 4. 日次業務割り当て（ガントチャート）
日ごとの詳細な業務スケジュールを作成します。

1. 「シフト作成・管理」画面で、編集したい日付（「1日」「2日」などのヘッダー）をクリックします。
2. 画面下部にその日のガントチャートが表示されます。
3. **業務の配置**:
   - 上部の業務ボタン（色付きボタン）から配置したい業務を選択します。
   - ガントチャート上の空いている時間帯をクリックすると、業務が配置されます。
   - **削除**: 配置済みの業務をクリックすると削除されます。
4. **コピー＆ペースト**:
   - 「コピー」ボタンを押すと、現在の配置パターンを一時保存します。
   - 別の日付を開き、「貼り付け」ボタンを押すと、同じ配置を適用できます。
5. **クリア**: 「クリア」ボタンで当日の配置をすべて消去します。

## 5. 印刷
作成したシフト表やガントチャートを印刷します。

- **月間シフト表**: 「シフト作成・管理」画面右上の「印刷」ボタンを押します。
- **日次ガントチャート**: ガントチャート表示エリア内の「印刷」ボタンを押します。
- **注意**: 印刷設定で「背景のグラフィック」を有効にすると、色付きで印刷されます。用紙は「A4横」を推奨します。

## 6. データの保存と復元
データは自動的にブラウザに保存されますが、機種変更時などはクラウド同期を利用します。

- **クラウド保存**: 管理画面の設定メニュー（実装状況による）からAPIキーを入力し、「クラウドへ保存」を実行します。
- **復元**: 新しい端末でAPIキーを入力し、「クラウドから読み込み」を実行します。
